# Vivid Dotfiles
## My Setup
* WM: AwesomeWM
* Editor: Neovim
* Prompt: Rofi
* Shell: zsh
* Terminal: alacritty


## Install
To install run all my dotfiles run `stow */` for specific dotfiles run `stow <dotfile>`
