# Dotfiles for \*NIX
## Arch Linux Branch
This repository contains the dotfiles for various programs I use(on my ArchLinux machine).

## Contents
* zsh
  * .zshrc -> the basic config
  * .oh-my-zsh/themes/bullet-train.zsh-theme -> modified themes
* neovim
  * vim-plug integration
  * Startup Menu -> startify
  * themed with palenight
  * ranger integration
  * fzf integration
  * Action-Menu/Keybind-menu -> whichkey

* XMonad
  * default terminal
  * default browser
  * changed modkey
  * emwh support
  * Layout
    * Spacing between windows
    * focus, normal window colors
    * modded workspaces
* Alacritty Configfiles
* Kitty
  * Themed -> dracula theme
  * Configured icat
* Ranger
  * Themed -> dracula theme
